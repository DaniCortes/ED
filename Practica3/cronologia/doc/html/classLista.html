template<class T>
struct Lista< T >

T.D.A. Lista.

Una instancia l del tipo de datos abstracto Lista es un objeto formado por un int que indica el número de datos, el NodoLista con el primer dato, el NodoLista con el último dato y un bool llamado init para controlar la inserción del primer T dato
