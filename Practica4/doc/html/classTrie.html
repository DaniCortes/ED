Descripción detallada

T.D.A. Árbol Trie.

Una instancia c del tipo de datos abstracto Trie es un árbol genérico donde cada nodo contiene una letra que constituye un prefijo de una palabra. Sirve para almacenar palabras. En la instancia únicamente se almacena explícitamente el Nodo raiz.
